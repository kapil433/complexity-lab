# Network Dynamics Simulations

Applying complexity science to understand emergent behavior in automotive market networks.

## Research Questions

1. How do dealer networks exhibit emergent clustering patterns?
2. What drives EV adoption cascades in Indian cities?
3. Network effects in charging infrastructure deployment

## Methods

- Agent-based modeling (Mesa framework)
- Network analysis (NetworkX)
- Diffusion models for technology adoption
- Power-law distributions in registration data

## Applications

- Predicting market tipping points for EVs
- Understanding MSIL dealer network resilience
- City-level EV adoption modeling
